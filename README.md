# Basic Operations with Numbers in Python

This project contains simple Python functions for basic number operations. Each function name clearly describes its purpose.

## Functions

- `calculate_average()` — calculates the average of given numbers.
- `minutes_to_hours()` — converts minutes into hours.
- `calculate_discount()` — calculates the discount amount for a given price and percentage.
- `get_last_digit()` — returns the last digit of a number.
- `split_number_digits()` — splits a 4-digit number into its individual digits.
- `calculate_rectangle_perimeter()` — calculates the perimeter of a rectangle given its sides.
- `square_number()` — returns the square of a given number.
